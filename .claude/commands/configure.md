# /configure

Modify the Melange configuration for: $ARGUMENTS

Read `.claude/skills/configure/SKILL.md` and follow its process exactly.

1. Classify the change as PROJECT (free) or UNIVERSAL (gated — requires written justification)
2. Identify all co-located files that need updating
3. If UNIVERSAL: stop and ask for justification before proceeding
4. Make all changes
5. Run the consistency verification gate and report DONE or BLOCKED

Do not make multiple structural changes in a single operation. One change at a time.
